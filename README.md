# Codictate

Codictate is a macOS app that transcribes your voice directly into any text field — no copy-pasting, no switching windows. Just press a shortcut, speak, and your words appear wherever your cursor is.

## Download

Go to [Releases](https://github.com/EmilLykke/codictate-releases/releases) and download the latest `.dmg` file.

| Channel | Description |
|---------|-------------|
| **Stable** | Recommended for most users |
| **Canary** | Latest features, updated more frequently |

## Installation

1. Download the `.dmg` for your channel
2. Open it and drag **Codictate** to your Applications folder
3. Launch it — macOS will ask for a few permissions the first time (Microphone, Input Monitoring, Accessibility)

## Requirements

- macOS 13 or later
- Apple Silicon (arm64)
